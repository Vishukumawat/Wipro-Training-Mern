# SmartLearn Course Enrollment Portal

A simple MERN stack project for managing courses and enrollments.  
This project is created for the Final Milestone mock Assessment.

#Project Overview

SmartLearn is an EdTech portal where learners can:

- View all available courses  
- Enroll in a selected course  
- Prevent duplicate enrollments  
- View enrolled courses in the UI  

The project consists of:

- Backend → Node.js, Express, MongoDB  
- Frontend→ React + Bootstrap  
- Testing → Mocha, Chai, SuperTest
