# FitTrack – Personal Fitness Training Portal

A simple MERN stack project for managing fitness programs and enrollments.  

# Project Overview

FitTrack is a fitness training portal where users can:

- View all available fitness programs  
- Enroll in a selected program  
- Prevent duplicate enrollments  
- View enrolled programs in the UI  

The project consists of:

- Backend → Node.js, Express.js, MongoDB  
- Frontend → React + Bootstrap  
- Testing → Mocha, Chai, SuperTest
